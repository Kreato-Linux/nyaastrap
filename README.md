# nyaastrap v2
The Kreato Linux build tool.

# Usage
Available commands;
`build-packages`
Builds packages.
`build-rootfs`
Builds a Kreato Linux rootfs according to the `.config` file. A simple one is included.

You can execute commands by `docker-compose up [COMMAND]`.
